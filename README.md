# 📚 사내 규정 검색기 v9.2

> **로컬 AI 기반 사내 규정 문서 검색 프로그램**  
> PyQt6 기반 현대적인 UI | 하이브리드 검색 | 증분 인덱싱 | 검색 결과 내보내기

---

## ✨ 주요 기능

### 🔍 하이브리드 검색 엔진
| 방식 | 설명 | 가중치 |
|------|------|--------|
| **벡터 검색** | HuggingFace 한국어 임베딩 모델로 의미 기반 검색 | 70% |
| **키워드 검색** | BM25 알고리즘으로 정확한 키워드 매칭 | 30% |

- 🔎 검색어 하이라이트 표시
- ⏱ 검색 소요 시간 표시
- 📥 검색 결과 내보내기 (TXT/CSV)
- 🕐 검색 히스토리 드롭다운
- 📊 매칭 점수 시각화 (색상 + 프로그레스 바)

### 📄 지원 문서 형식
| 형식 | 확장자 | 비고 |
|------|--------|------|
| 텍스트 | `.txt` | 모든 인코딩 자동 감지 |
| Word | `.docx` | 테이블 내용 포함 추출 |
| PDF | `.pdf` | 텍스트 기반만 (이미지 PDF 제외) |

### ⚡ 성능 최적화
- **증분 인덱싱**: 변경된 파일만 재처리
- **캐시 시스템**: 처리 결과 로컬 저장으로 빠른 재로드
- **배치 처리**: 메모리 효율적 문서 처리
- **GPU 가속**: CUDA 사용 가능 시 자동 활용

### 🎨 사용자 인터페이스
- PyQt6 기반 현대적 다크 테마
- 3개 탭 구성: **검색** | **파일** | **설정**
- 파일 테이블: 정렬 가능, 더블클릭으로 원본 열기
- 통계 패널: 파일 수, 청크 수, 총 용량
- 폰트 크기 조절 슬라이더

## 📖 사용 방법

### Step 1: 모델 로딩
프로그램 시작 시 AI 임베딩 모델이 자동 로드됩니다.  
첫 실행 시 모델 다운로드로 2~5분 소요될 수 있습니다.

### Step 2: 폴더 선택
`📂 폴더 열기` 버튼을 클릭하여 규정 문서 폴더를 선택합니다.

### Step 3: 문서 처리
선택된 폴더의 문서가 자동으로 분석되고 인덱싱됩니다.
- 처리 상태는 진행률 표시줄로 확인
- 완료 후 **파일** 탭에서 처리 결과 확인
- 처리 실패 파일이 있으면 알림으로 표시

### Step 4: 검색
검색어를 입력하고 Enter 또는 `🔍 검색` 버튼을 클릭합니다.
- 결과 개수 조절 가능 (1~10개)
- 하이브리드 검색 On/Off 가능
- 검색 결과 내보내기 (📥 버튼)
